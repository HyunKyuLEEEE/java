조건문
프로그램 수행 흐름을 바꾸는 역활을 하는 제어문 중 하나로 조건에 따라 다른 문장이 수행되도록 함
조건식엔 참 또는 거짓이 와야함

조건식의 결과 값이 true면 중괄호 안의 내용 실행 false면 실행하지 않음
if : 반드시 필요, 조건문을 시작, 1개
else if : 0개 이상 (있을 수도 있고 없을 수도 있으면 여러개 가 있을 수 있음)
else : 0개 또는 1개 (없을 수도 있고 있으면 최대 1개 )

if, else if에만 소괄호가 들어간다. else에는 들어가지 않는다.

switch문
조건식 하나로 많은 경우의 수를 처리할 때 사용하며 이때 조건식의
결과는 정수 또는 문자, 문자열

조건식의 결과값과 일치하는 case문으로 이동
dafault문은 일치하는 case문이 없을 때 수행(=else)


반복문
프로그램 수행 흐름을 바꾸는 역할을 하는 제어문 중 하나로 
특정 문장들을 규칙적인 작업을 반복해서 수행하도록 함
for(1.초기화; 2.5.8.조건식; 4.7.증감식){
	3.6.실행문;
}
초기화 : 조건식 또는 실행문에서 사용하는 변수를 초기화
	 for문 시작 시 1번만 실행, 생략 가능(초기화 할게 없을 시)
조건식 : 거짓이면 반복문을 종료, 참이면 반복문을 실행
	 생략 가능, 생략하면 무조건 참
증감식 : 조건식에서 사용하는 변수를 증가/감소시켜서 반복문 횟수를 결정
	생략 가능
- 초기화 + 조건식 + 증감식 => 반복 횟수를 결정할 수 있다.
- 실행문 : 규칙적인 작업
